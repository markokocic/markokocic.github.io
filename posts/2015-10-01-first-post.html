First Post

This is my first post!

I have been playing around with the idea of finally updating my "Home Page". Before, it was only my CV hosted on the Google AppEngine, but it became kind of annoying to update it regularly.

I was thinking about setting up something simpler. Didn't like the idea of having to set up the whole full featured blogging platform like WordPress or similar. I wanted something simple, something that I could easily update, preferably using any text editor, and which wouldn't require more than Apache web server to host it. Thus, I started looking at static website generators.

When I came around the Cryogen, I was sold. It looked exactly what I needed, so I decided to start my personal "site" using it.
