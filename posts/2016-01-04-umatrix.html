uMatrix and uBlock

uMatrix and uBlock Origin

Since a few months I started using uBlock Origin as a privacy plugin for Firefox. Although it works good, after some time it start to be annoying having to constantly tweak it. It's ok that it tries to be user friendly, but sometimes I just feel like too much tweaking is required to get a balance between privacy and having non-broken web browsing experience.

Then, I came across a new Firefox plugin by the same author as uBlock. It's called uMatrix. It allows for a nice matrix like UI to define the blocking and allow rules. It has sensible defaults, which provide nice tradeofs out of the box with minimal tweaking.

Right now, I'm using uMatrix as my only ad blocker and tracker blocker tool on all my browsers. If you haven't tried it, just give it a go, you may never look back.

The only drawback is that it doesn't work on the Firefox version for Android, at least not yet.
